Dear Users of Ledmac and Ledpar.

As you know, I maintain Ledmac and ledpar one year ago.

In the previous day, I made some important modification to allow more easy customization of critical footnote. I hope to publish a new release in next day, with the helping of Daniel Gasparotto for documentation. This release will be the last of the series 0.X.

It will include these commands, where X is a series : 
- \lemmaseparator[X]{sep}, to change separator between lemma and note.
- \beforelemmasepararor[X]{length}, to add blank before lemma separator.
- \afterlemmaseparator[X]{length} to add blanck after lemma separator.
- \nolemmaseparator[X] to disable lemma separator.
- \nonumberinfootnote[X] to disable line number in critical footnote.
- \beforenumberinfootnote[X]{length} to add space before line number.
- \afternumberinfootnote[X]{length} to add space after line number.
- \afternote[X]{length} to add space after note (in paragraphed footnote).
- \parafootsep[X]{sep} to change separator betwen note (in paragraphed footnote).
- \numberonlyfirstinline[X] to print number of line only the first time for each line.
- \symplinenumber[X] to have a symbol for line number when it the same line (not yet implemented, but will be). It will remplace \symplinenum (but which will be kept for ascendant compatibility).. 

All length can be stretchable, and default value are the actual value.


I think there is no break of compatibility. But you must have the package etoolbox and xargs. These packages are install in the recent versions of major TeX distribution (MacTeX, TeXLive and MikTeX). If some break appear, send me an email. 

Thank to ousia for idea, and to Daniel for helping.

The Roadmap for next time is :
- release 1.0 : new command to created a new series. Just \newfootseries{X} to create X series, with all personalisables functions. I think the X command will be deleted, but put in a "attic" for people which used them. It will be occasion to simplify code. It's need if you want I maintain package in time.
- release 1.1 : footnote by section / in parallel.
- release 1.2 : problem with long footnote.

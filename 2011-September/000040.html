> Summary description: Typeset scholarly editions in parallel texts.
> License type: lppl
> 
> Announcement text: 
> ----------------------------------------------------------------------
> In some languages, hanging indents of verses must start with a symbol. For 
> example, in the french typographie, the symbol is [.
> 
> This new version of ledmac and ledpar provides a method to insert it 
> automatically: The user has to redefine the macro \hangingsymbol.
> 
> For example, in the case of the french typography, put this in your file :
> \renewcommand{\hangingsymbol}{[\,}.
> 
> Note that this new function was tested, but some bugs may still exist: 
> please provide feedback via github.
>

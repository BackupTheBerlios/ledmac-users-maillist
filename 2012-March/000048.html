> Dear ledmac-users,
> 
> I want to typeset an edition of a number of small glosses in critical edition. First I thought simply to use the minipage-environmen of Latex, because most of the glosses are very short. But for a better fitting of the text on the book pages, it seems advisable to use ledmacs ledgroupsized environment, because it allows pagebreaks. But there are two problems to be solved:
> 
> (1) the apparatuses should be centered (like the main text) - a feature which is not supportet by default (cf. manual p. 9)
> (2) the line numbers should be placed near the text (by default they are placed in the margin of the page)
> 
> Would be nice, if somebody could help me.
> 
> B. Koelbl

>> Objet : Un livre sur (Xe)LaTeX appliqué aux sciences humaines
>> 
>> Chers amis, collègues, professeurs,
>> 
>> j'ai parlé à certains d'entres vous de LaTeX.
>> 
>> LaTeX est un logiciel de composition de texte offrant de nombreuses possibilités, dont :
>> 	- gestion d'une bibliographie
>> 	- gestion d'éditions critiques.
>> 	- gestion de traduction d'un texte en //.
>> 
>> Je l'ai utilisé notamment pour composer mon mémoire.
>> 
>> Il n'existe cependant pas de manuel d'introduction à LaTeX présentant les outils utiles aux sciences humaines, notamment pour les conventions bibliographiques, …  jusqu'à aujourd'hui.
>> 
>> J'ai en effet le plaisir de vous annoncer que mon Livre intitulé (Xe)LaTeX appliqué aux sciences humaines vient de sortir aux éditions Atramenta.
>> 
>> Il est disponible en vente (pour le modeste prix de 13€ pour 270 pages) et en téléchargement sur le site de l'éditeur (pas de ventes en librairie) : http://www.atramenta.net/books/latex-sciences-humaines/79. Pour les Strasbourgeois, je vais envoyer une version papier à la RHPR. Si certain veulent acheter leur propre exemplaire, je dispose de réductions.

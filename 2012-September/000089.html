> License type: lppl

> Announcement text: 
> ----------------------------------------------------------------------
> 
> Eledmac and eledpar are successors of ledmac / ledpar. 
> 
> ledarab has no successor, but people should use XeLaTeX possibility.
> 
> The changes in eledmac and eledpar with respect to ledmac / ledpar are :
> - new generic command \newseries to add a new series of notes (critical, familiar, end)
> - new commands to easily customize the display of notes.
> - default maxchunks (in eledpar) at 5120. 
> - use of xargs, etoolbox and etex to simplify code.
> - code refactoring
> 
> For people who migrate from eledmac to eledpar, it's possible they have some 
> changes to make. These changes are explained in Appendix A of eledmac doc.
>

> I'm typesetting an edition where we decided that we want to print two
> manuscript-pages per printed page, as the ms has a rather small
> format. The line-breaks are supposed to agree with the
> manuscript's. The only problem is that this renders the apparatus
> ambigous, as every linenumber apprears twice per page. Thus I'd like
> to insert 1r, 1v, 2r,... directly in front of each line-number in the
> apparatus.

I'm sorry not to have a complete solution to your problem.  But as no
one answered your question, I'll do a wild guess.

If I understand you accurate, you want for both manuscript page only one
apparatus.  So you probably do have a two-column setup.

You have to change the code the lemmata are generated, so an additional
info about the column your in is stored.  And you have to change the way
the footnotes are printed.

So I guess you have to read the sections of the well-documented code in
the ledmac manual.

If you need further help, a working minimal example would be a good
starting point.
